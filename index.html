Clash of Code

Clash of Code is a annual coding compition organized by the alumni association of National Institute of Technology Arunchal Pradesh.The aim of the event is to provide a platform for the students to practice their coding skills and compete with each other, it also aims to increase the interaction between the alumni as the students of the university

The first clash of code event was organized on Nov 2016.
